# Agentic Developer Portal

A high-performance, AI-native developer cockpit built with **React**, **TypeScript**, and **Redux Toolkit**. This portal demonstrates how developers can orchestrate AI agents for repository maintenance, refactoring, and health monitoring.

## 🚀 Key Features

- **Repository Insights:** Real-time health scores and tech-stack metadata for microservices.
- **Agentic Workflow:** A simulated, asynchronous AI execution engine using Redux Thunks.
- **Streaming Terminal:** A real-time log viewer to monitor agent "reasoning" and progress.
- **Responsive UI:** Built with Tailwind CSS for a sleek, "developer-first" aesthetic.

## 🛠️ Tech Stack

- **Framework:** React 18 (Vite)
- **State Management:** Redux Toolkit (RTK)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Language:** TypeScript
